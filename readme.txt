== Description ==

Are you having trouble finding great content for your blog/website?
Content Party is a content exchange service platform. Our Content Party plugin can help you get the best and authorized contents in only three simple steps. This plugin seamlessly integrates using the Content Party API and by syncing with your WordPress backend. 
你還在煩惱稿量不足、品質不優的問題嗎？此外掛由 Content Party 內容交換服務平台開發，只要簡單 3 步驟，就能幫你快速地取得優質且合法授權的文章內容。所有取用過程都在你的 WordPress 後台完成，再也不用於兩個網站間來回轉換。

**Main feature**

* Content Party provides the most extensive, high-quality content, with convenient searching and sorting capability.
* Featured within top twenty popular keywords, which helps you to find the popular topic with ease.
* We have a variety of content, such as technology, travel, health, etc.
* Simple and easy to use.
 
PS: We will ask you to login your Content Party account when you first launch this plugin, since all the content in Content Party are authorized, users must login their account to use these content legitimately. If you don’t have Content Party’s account yet you can [register here](https://contentparty.org/user_reg/register).

**外掛特色**
* 提供最豐富的優質內容，搭配便利的搜尋與排序功能，好文輕鬆到手。
* 精選二十大熱門關鍵字排行，當紅主題，一鍵找到！ 
* 匯集各方的優質供稿媒體，3C 科技、旅遊、健康...，通通有！
* 最便捷的取用方式，3步驟完成文章取用，批次取用多篇也 ok！

備註：因為 Content Party 內的文章，皆為合法授權內容，取用者皆須申請帳號後才能合法取用。當你第一次使用此外掛時，會請你登入 Content Party 帳號。若尚未擁有帳號，歡迎前往 [Content Party](https://contentparty.org/user_reg/register) 註冊。

== Installation ==

**From your WordPress dashboard**
1. Visit 'Plugins > Add New'
2. Search ‘Content Party’
3. Install and activate Content Party from your Plugins page. 

**從你的 WordPress 後台**

1. 於左側選單中選擇「外掛」>「安裝外掛」
2. 搜尋「Content Party」
3. 點選「立即安裝」
4. 安裝完成後，點選頁面下方的「啟用外掛」

== Frequently Asked Questions ==

**Can I edit the content I selected?**
Base on the respect for the original work, you can only edit the title, summary and tags.

**我可以更改取得文章的內容嗎？**
基於尊重原作的前提下，取用的文章僅提供修改標題，內文無法修改，摘要與 tag 則可由你自由填寫。保留內容原貌與廣告，便是給予原作的鼓勵與回饋。
